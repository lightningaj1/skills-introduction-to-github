QGIS Export Instructions
GeoJSON Export:
  1. Select your layer in QGIS
  2. Right-click → Export as → GeoJSON
  3. Click Upload on the Deposits or Claims tab
CSV Export:
  1. Select your layer in QGIS
  2. Right-click → Export as → CSV
  3. Ensure latitude/longitude columns are named correctly
  4. Click Upload above
Required Data Fields
Mineral Deposits:
  • ✓ name - Deposit name
  • ✓ latitude - Coordinate
  • ✓ longitude - Coordinate
  • ○ country - Country name
  • ○ region - Region/province
  • ○ status - Active/Prospect/Historical
  • ○ reserves - Estimated reserves (tonnes)
  • ○ grade - Average grade (%)
Mining Claims:
  • ✓ claim_id - Unique claim identifier
  • ✓ latitude - Coordinate
  • ✓ longitude - Coordinate
  • ○ company - Company name
  • ○ area_hectares - Claim area
  • ○ status - Active/Inactive/Expired
Tips
  • Column names are case-insensitive (Name, NAME, name all work)
  • Latitude/Longitude must be numeric values (decimal degrees)
  • Duplicate entries (same name and location) are automatically skipped
  • You can import from multiple QGIS projects - they will be merged
  • Use the "Download Template" buttons to see exact expected format
  • Maps will update automatically after import
